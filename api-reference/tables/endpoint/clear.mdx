<Warning>
**DEPRECATED**: This endpoint has been deprecated and will be removed on **March 1, 2026**.
Please use the new endpoint: [`POST /v1/uploads/:namespace/:table_name/clear`](./uploads-clear)

See the [Migration Guide](./migration) for details on migrating to the new endpoints.
</Warning>

The `Clear` endpoint removes all the data in the specified table, but does not delete the table.

<RequestExample>

```bash curl
curl --request POST \
  --url https://api.dune.com/api/v1/table/my_user/interest_rates/clear \
  --header 'X-DUNE-API-KEY: <x-dune-api-key>'
```


```python Python
import requests

url = "https://api.dune.com/api/v1/table/my_user/interest_rates/clear"

headers = {
    "X-DUNE-API-KEY": "<x-dune-api-key>"
}
```
