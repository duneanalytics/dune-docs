## Key Features

<CardGroup cols={3}>
  <Card title="Advanced Filtering" icon="filter" href="/api-reference/executions/filtering">
    **Unique to Dune** - Filter results directly in API calls without modifying queries. Use SQL-like WHERE clauses to refine data on the fly.
    ```python
    dune.get_latest_result(
      query=3567562,
      filters="block_time > '2024-03-01'"
    )
    ```
  </Card>
  
  <Card title="Execute Raw SQL" icon="code" href="/api-reference/executions/endpoint/execute-query">
    **New** - Run arbitrary SQL queries via API, not just saved queries. Perfect for dynamic analysis and automated workflows.
    ```python
    dune.execute_sql(
      "SELECT * FROM dex.trades 
       WHERE block_time > now() - interval '1 day'"
    )
    ```
  </Card>
  
  <Card title="Query Pipelines" icon="diagram-project" href="/api-reference/executions/endpoint/execute-query">
    Execute entire query lineages including materialized views and dependent queries in a single API call.
    ```python
    dune.execute_pipeline(
      pipeline_id=12345
    )
    ```
  </Card>
</CardGroup>

## Quick Start with SDKs

Our SDKs handle authentication, polling, pagination, and error handling automatically. The new `execute()` function manages the entire workflow for you.

<CardGroup cols={3}>
  <Card title="Python" icon="python" href="/api-reference/overview/sdks">
    ```bash
    pip install dune-client
    ```
    ```python
    from dune_client import DuneClient
    
    dune = DuneClient(api_key="your-key")
    results = dune.execute(query_id=3493826)
    ```
    
    [Full Python documentation →](/api-reference/overview/sdks)
  </Card>
  
  <Card title="TypeScript" icon="js" href="/api-reference/overview/sdks">
    ```bash
    npm install @duneanalytics/client
    ```
    ```typescript
    import { DuneClient } from '@duneanalytics/client';
    
    const dune = new DuneClient('your-key');
    const results = await dune.execute({ queryId: 3493826 });
    ```
    
    [Full TypeScript documentation →](/api-reference/overview/sdks)
  </Card>
  
  <Card title="Go" icon="golang" href="/api-reference/overview/sdks">
    ```bash
    go get github.com/duneanalytics/duneapi-client-go
    ```
    ```go
    client := duneapi.New("your-key")
    results := client.Execute(3493826)
    ```
    
    [Full Go documentation →](/api-reference/overview/sdks)
  </Card>
</CardGroup>

<Note>
  **New:** All SDKs now include an `execute()` function that handles polling and pagination automatically. No more managing multiple API calls!
</Note>
