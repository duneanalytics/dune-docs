---
title: 'Get Latest Query Result'
openapi: 'GET /v1/query/{query_id}/results'
---

You must pass parameter query_id. 

Returns the latest execution id and results (in JSON) of that latest run, regardless of the job id/run or if it is run in the app or the api. 

The query specified must either be public or a query you have ownership of (you or a team you belong to have ownership).

<Tip>  
- This endpoint does NOT trigger an execution but does [consume credits through datapoints](https://dune.com/pricing).
- Read more on [Filtering](../filtering), [Sorting](../sorting), and [Sampling](../sampling) to learn more about flexibly grabbing query results. 
- Read more on [Pagination](../pagination) to get the most out of the API and handle large results.
</Tip>

<Info> 
- Dune internally has a maximum query result size limit (which currently is 8GB, but subject to increase in the future). If your query yields more than 8GB of data, the result will be truncated in storage. In such cases, pulling the result data (using pagination) but without specifying `allow_partial_results` set to true will trigger an error message: "error": "Partial Result, please request with 'allows_partial_results=true'". If you wish to retrieve partial results, you can pass the parameter `allow_partial_results=true`. But please make sure you indeed want to fetch the truncated result.
</Info>

## Examples and Tutorials

Welcome to the Examples and Tutorials section! Here, you'll find a collection of code examples and step-by-step tutorials to help you get started with getting the latest query result.

### Step-by-Step Tutorial: Getting the Latest Query Result

#### Step 1: Define the Query

Before you can get the latest query result, you need to define the query with its parameters. This includes specifying the query ID and any parameters required for the query.

**Example using Python SDK:**

```python
import dotenv, os, json
from dune_client.types import QueryParameter
from dune_client.client import DuneClient
from dune_client.query import QueryBase

# change the current working directory where .env file lives
os.chdir("/Users/abc/local-Workspace/python-notebook-examples")
# load .env file
dotenv.load_dotenv(".env")
# setup Dune Python client
dune = DuneClient.from_env()

# define the query to run
query = QueryBase(
    name="Sample Query",
    query_id=1215383,
    params=[
        QueryParameter.text_type(name="TextField", value="Word"),
        QueryParameter.number_type(name="NumberField", value=3.1415926535),
        QueryParameter.date_type(name="DateField", value="2022-05-04 00:00:00"),
        QueryParameter.enum_type(name="ListField", value="Option 1"),
    ],
)
```

#### Step 2: Execute the Query

Once you have defined the query, you can execute it by making a POST request to the execute endpoint.

**Example using cURL:**

```bash
curl -X POST -H x-dune-api-key:{{api_key}} "https://api.dune.com/api/v1/query/{{query_id}}/execute" -d '{"query_parameters": {"param1":24}, "performance": "large"}'
```

**Example using Python SDK:**

```python
result = dune.execute_query(query = query # pass in the query we just defined
          , performance = "large" # define engine query is run on, default is "medium"
)
print(result)
```

**Example using Python Requests:**

```python

```

#### Step 3: Retrieve the Latest Query Result

Once you have executed the query, you can retrieve the latest query result by making a GET request to the results endpoint.

**Example using cURL:**

```bash
curl -X GET -H x-dune-api-key:{{api_key}} "https://api.dune.com/api/v1/query/{{query_id}}/results"
```

**Example using Python SDK:**

```python
import dotenv, os, json

# change the current working directory where .env file lives

# load .env file

result = dune.get_latest_result(job_id = '01HM7F50BXZVPHJNWJQM6GMS1M') # pass in the execution_id

```
