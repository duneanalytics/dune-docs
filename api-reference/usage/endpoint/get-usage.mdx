title: "Get Usage"
openapi: "GET /v1/usage"
description: "Retrieve customer usage data for billing periods"
---

## Description

Retrieves customer usage data including credits, queries, dashboards, and storage. Returns the current billing period when no timeframe is specified.


## Pricing

This is a metadata endpoint and does not consume credits.

<RequestExample>

```bash cURL
curl -X GET "https://api.dune.com/api/v1/usage?start_date=2025-01-01&end_date=2025-02-01" \
  -H "X-DUNE-API-KEY: YOUR_API_KEY"
```

```python Python
import requests

url = "https://api.dune.com/api/v1/usage"
params = {
    "start_date": "2025-01-01",
    "end_date": "2025-02-01"
}
headers = {"X-DUNE-API-KEY": "YOUR_API_KEY"}

response = requests.get(url, params=params, headers=headers)
print(response.json())
```

```javascript JavaScript
const url = 'https://api.dune.com/api/v1/usage';
const params = new URLSearchParams({
  start_date: '2025-01-01',
  end_date: '2025-02-01'
});

const response = await fetch(`${url}?${params}`, {
  headers: {
    'X-DUNE-API-KEY': 'YOUR_API_KEY'
  }

const data = await response.json();
console.log(data);
```
