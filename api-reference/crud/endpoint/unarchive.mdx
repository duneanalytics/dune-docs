---
title: "Unarchive Query"
openapi: "POST /v1/query/{queryId}/unarchive"
---

<Note>
  You need a [Plus plan](https://dune.com/pricing) or higher to use CRUD
  endpoints.
</Note>

<Tip>
  In Dune context, delete action is replaced by archive as deletion of queries
  is not possible.
</Tip>

<RequestExample>

```bash cURL
curl --request GET \
  --url https://api.dune.com/api/v1/query/{queryId}/unarchive \
  --header 'X-DUNE-API-KEY: <x-dune-api-key>'
```
