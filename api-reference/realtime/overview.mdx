Sim APIs are a dedicated, high-performance suite of developer-focused APIs by Dune, designed for building realtime, multichain apps with exceptional developer experience. Sim APIs offer fast and reliable access to EVM and SVM blockchain data.

**Sim APIs are an upgraded version of what was previously known as Dune Echo.** They offer improved performance and a broader set of features for developers.
