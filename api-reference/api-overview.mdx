Execute queries, manage data, and access blockchain analytics programmatically via REST endpoints.

The Dune Analytics API provides programmatic access to execute queries, retrieve results, manage queries and data uploads, and monitor your usage—all through REST endpoints.

Execute saved queries on demand, run arbitrary SQL, upload custom datasets, and retrieve results in structured formats for your applications.

## Available Endpoints

### Executions and Results

Execute saved queries and retrieve results in various formats

The executions and results endpoints facilitate the execution of saved queries, retrieval of results in JSON or CSV format, and management of query executions.

[View Executions and Results →](/api-reference/executions/execution-object)

### Query Management

Create, update, and manage your Dune queries programmatically

These endpoints help you programmatically create and manage DuneSQL queries via the API, ideal for integration with GitHub or other CI/CD pipelines.

[View Query Management →](/api-reference/queries/endpoint/query-object)

### Data Uploads

Upload and manage custom datasets as queryable tables

Upload CSV files or programmatically create and manage tables to combine your own data with Dune's blockchain data.

[View Data Uploads →](/api-reference/tables/endpoint/overview)

### Materialized Views

Create and refresh materialized views for improved query performance

Manage materialized views to store query results as queryable tables, enabling faster access to frequently-used data transformations.

[View Materialized Views →](/api-reference/materialized-views/overview)

### Account Management

Monitor API usage and credit consumption

Track your account's resource consumption, including credits consumed, query executions, and storage utilization for billing transparency.

[View Account Management →](/api-reference/usage/overview)
