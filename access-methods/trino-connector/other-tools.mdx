# Other Trino-Compatible Tools

Connect any tool that supports Trino/Presto protocol to Dune.

## Overview

Any tool that supports the Trino (formerly Presto) protocol can connect to Dune's data. This includes:

- Jupyter Notebooks (via trino-python-client)
- Apache Superset
- Redash
- Mode Analytics
- And many more
