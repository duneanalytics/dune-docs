Access Dune's blockchain data from your favorite BI tools and clients using the DuneSQL Trino/Presto connector.

## Overview

The Trino Connector provides **read-only access** to Dune's comprehensive blockchain data warehouse. Connect Dune with popular tools like Hex, Metabase, DBeaver, and any Trino, Presto, or Starburst-compatible application for seamless data analysis.

<Note>
The Trino Connector is available on Plus and Enterprise plans. [View pricing](https://dune.com/pricing) for details.
</Note>

## Key Features

- **Standard Trino Protocol**: Compatible with any tool that supports Trino/Presto/Starburst
- **Full Data Access**: Query all of Dune's blockchain data, including raw, decoded, and curated tables
- **Native SQL**: Use familiar SQL syntax in your preferred environment
- **Real-time Data**: Access the same fresh data available in the Dune web app
