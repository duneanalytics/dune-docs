Connect your favorite BI tools, notebooks, and SQL clients directly to Dune's data warehouse using the standard Trino protocol.

## Overview

The Trino Connector provides **read-only access** to Dune's comprehensive blockchain data warehouse. Query the same data that powers thousands of Dune dashboards directly from your existing data infrastructure and BI tools.

<Note>
The Trino Connector is available on Plus and Enterprise plans. [View pricing](https://dune.com/pricing) for details.
</Note>

## Key Features

- **Standard Trino Protocol**: Compatible with any tool that supports Trino/Presto
- **Full Data Access**: Query all of Dune's blockchain data, including raw, decoded, and curated tables
- **Native SQL**: Use familiar SQL syntax in your preferred environment
- **Real-time Data**: Access the same fresh data available in the Dune web app
