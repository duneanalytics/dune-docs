---
title: 'Supported Chains'
openapi: 'GET /beta/balance/chains'
---

The chains endpoint lists the supported blockchains for the Token Balances endpoint.

<Note>
  - URL may change when we move from /beta/ to /v1/. This is planned in Q3 2024.
</Note>

# Column Descriptions

| Column                    | Description                                        | Type                |
|---------------------------|----------------------------------------------------|---------------------|
| name                      | Canonical blockchain name                          | string              |
| id                        | Chainid of blockchain                              | integer             |
| supported_assets          | The supported asset types                          | []string            |
| default                   | Whether the chain is in the list of default chains | bool                |
