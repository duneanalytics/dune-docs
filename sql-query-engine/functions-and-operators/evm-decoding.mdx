# EVM Decoding Functions

Decode raw EVM data within your SQL queries.

## abi_decode

Decode ABI-encoded data.

```sql
SELECT abi_decode(
    'transfer(address,uint256)',
    data
) as decoded
FROM ethereum.logs
```

## abi_encode

Encode data according to ABI specification.

```sql
SELECT abi_encode('transfer(address,uint256)', address, amount)
```
