# Chain Utility Functions

Functions specific to blockchain data.

## get_chain_id

Get the chain ID for a blockchain name.

```sql
SELECT get_chain_id('ethereum') -- Returns 1
```

## get_chain_name

Get the chain name for a chain ID.

```sql
SELECT get_chain_name(1) -- Returns 'ethereum'
```
