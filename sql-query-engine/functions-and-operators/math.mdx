# Math Functions

Mathematical operations and functions.

## Basic Math

| Function | Description |
|----------|-------------|
| `abs(x)` | Absolute value |
| `ceil(x)` | Round up |
| `floor(x)` | Round down |
| `round(x, d)` | Round to d decimals |
| `power(x, y)` | x to the power y |
| `sqrt(x)` | Square root |
| `log10(x)` | Base-10 logarithm |
| `ln(x)` | Natural logarithm |

## Statistical

| Function | Description |
|----------|-------------|
| `stddev(x)` | Standard deviation |
| `variance(x)` | Variance |
