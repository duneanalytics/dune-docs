# ORDER BY

Sort results.

## Ascending (Default)

```sql
SELECT * FROM ethereum.transactions
ORDER BY block_time
```

## Descending

```sql
SELECT * FROM ethereum.transactions
ORDER BY block_time DESC
```

## Multiple Columns

```sql
SELECT * FROM ethereum.transactions
ORDER BY block_number DESC, tx_index ASC
```

## Nulls Handling

```sql
SELECT * FROM table
ORDER BY column NULLS FIRST
-- or
ORDER BY column NULLS LAST
```
