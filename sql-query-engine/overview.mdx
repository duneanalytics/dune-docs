# SQL Query Engine

DuneSQL is a Trino-based SQL engine optimized for querying blockchain data at scale.

## What is DuneSQL?

DuneSQL is built on Trino (formerly PrestoSQL) with custom extensions for blockchain data types and operations. It provides familiar SQL syntax with specialized functions for working with addresses, transactions, and decoded data.
