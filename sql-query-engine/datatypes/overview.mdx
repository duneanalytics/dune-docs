# Data Types

Data types available in DuneSQL.

## Numeric Types

| Type | Description |
|------|-------------|
| `integer` | 32-bit integer |
| `bigint` | 64-bit integer |
| `double` | 64-bit floating point |
| `decimal` | Arbitrary precision |
| `uint256` | 256-bit unsigned integer (for blockchain values) |

## String Types

| Type | Description |
|------|-------------|
| `varchar` | Variable-length string |
| `varbinary` | Variable-length binary (for addresses, hashes) |

## Date/Time Types

| Type | Description |
|------|-------------|
| `date` | Calendar date |
| `timestamp` | Date and time |

## Array and Map Types

| Type | Description |
|------|-------------|
| `array` | Ordered collection |
| `map` | Key-value pairs |
| `row` | Structured record |
