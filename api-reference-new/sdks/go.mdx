# Go SDK

The official Go client for the Dune API.

## Installation

```bash
go get github.com/duneanalytics/dune-go
```

## Quick Start

```go
package main

import (
    "fmt"
    dune "github.com/duneanalytics/dune-go"
)

func main() {
    client := dune.NewClient("YOUR_API_KEY")
    
    result, err := client.ExecuteQuery(1234)
    if err != nil {
        panic(err)
    }
    
    fmt.Println(result.Rows)
}
```
