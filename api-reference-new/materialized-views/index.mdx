# Materialized Views

Create, manage, and refresh materialized views programmatically.

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | [Get](/api-reference/materialized-views/get) | Get view details |
| POST | [Create/Upsert](/api-reference/materialized-views/create) | Create or update view |
| DELETE | [Delete](/api-reference/materialized-views/delete) | Delete view |
| GET | [List](/api-reference/materialized-views/list) | List your views |
| POST | [Refresh](/api-reference/materialized-views/refresh) | Refresh view data |
