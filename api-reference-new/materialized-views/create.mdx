# Create Materialized View

Create a new materialized view or update an existing one.

## Endpoint

`POST /api/v1/materialized-view`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| name | string | Yes | View name |
| query_sql | string | Yes | SQL query |
| refresh_interval | string | No | Refresh schedule |

```json
{
  "name": "my_view",
  "query_sql": "SELECT date_trunc('day', block_time) as day, SUM(amount_usd) as volume FROM dex.trades GROUP BY 1"
}
```

## Example Request

```bash
curl -X POST "https://api.dune.com/api/v1/materialized-view" \
  -H "X-Dune-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "my_view", "query_sql": "SELECT ..."}'
```
