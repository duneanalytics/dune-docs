# List Materialized Views

Retrieve a list of your materialized views.

## Endpoint

`GET /api/v1/materialized-view`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Example Request

```bash
curl "https://api.dune.com/api/v1/materialized-view" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```
