# Get Materialized View

Retrieve details of a materialized view.

## Endpoint

`GET /api/v1/materialized-view/{name}`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| name | string | Yes | The materialized view name |

## Example Request

```bash
curl "https://api.dune.com/api/v1/materialized-view/my_view" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```
