# Pagination

Handle large result sets with pagination.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| limit | integer | Max rows to return |
| offset | integer | Rows to skip |

## Example

```bash
# Get first 100 rows
curl "https://api.dune.com/api/v1/execution/{id}/results?limit=100&offset=0" \
  -H "X-Dune-API-Key: YOUR_API_KEY"

# Get next 100 rows
curl "https://api.dune.com/api/v1/execution/{id}/results?limit=100&offset=100" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```

## Response Metadata

The response includes metadata about total rows available.
