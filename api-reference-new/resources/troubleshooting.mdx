# Troubleshooting

Common issues and their solutions.

## Common Errors

### 401 Unauthorized

**Cause:** Invalid or missing API key

**Solution:** Verify your API key is correct and included in the `X-Dune-API-Key` header.

### 402 Payment Required

**Cause:** Credit limit exceeded

**Solution:** Check your usage and upgrade your plan if needed.

### 403 Forbidden

**Cause:** No access to the requested resource

**Solution:** Verify you have access to the query or table.

### 404 Not Found

**Cause:** Resource doesn't exist

**Solution:** Verify the query ID, table name, or execution ID is correct.

### 429 Too Many Requests

**Cause:** Rate limit exceeded

**Solution:** Implement backoff and retry logic. See [Rate Limits](/api-reference/resources/rate-limits).

### 500 Internal Server Error

**Cause:** Server-side error

**Solution:** Retry after a short delay. If persistent, contact support.
