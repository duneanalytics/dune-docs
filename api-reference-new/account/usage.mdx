# Get Usage

Retrieve your API usage statistics and credit consumption.

## Endpoint

`POST /api/v1/account/usage`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| start_date | string | No | Start date (ISO 8601) |
| end_date | string | No | End date (ISO 8601) |

## Example Request

```bash
curl -X POST "https://api.dune.com/api/v1/account/usage" \
  -H "X-Dune-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"start_date": "2024-01-01", "end_date": "2024-01-31"}'
```

## Response

```json
{
  "credits_used": 1500,
  "credits_remaining": 8500,
  "executions_count": 250,
  "data_points_returned": 1000000
}
```
