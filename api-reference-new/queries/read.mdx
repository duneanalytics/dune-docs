# Read Query

Retrieve details of a saved query.

## Endpoint

`GET /api/v1/query/{query_id}`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| query_id | integer | Yes | The query ID |

## Example Request

```bash
curl "https://api.dune.com/api/v1/query/1234" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```

## Response

```json
{
  "query_id": 1234,
  "name": "My Query",
  "description": "Query description",
  "query_sql": "SELECT * FROM ...",
  "is_private": false,
  "is_archived": false,
  "created_at": "2024-01-01T00:00:00Z",
  "updated_at": "2024-01-01T00:00:00Z"
}
```
