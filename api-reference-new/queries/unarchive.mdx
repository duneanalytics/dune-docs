# Unarchive Query

Restore a previously archived query.

## Endpoint

`POST /api/v1/query/{query_id}/unarchive`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| query_id | integer | Yes | The query ID to unarchive |

## Example Request

```bash
curl -X POST "https://api.dune.com/api/v1/query/1234/unarchive" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```
