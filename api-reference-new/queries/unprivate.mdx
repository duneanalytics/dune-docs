# Make Query Public

Make a private query public (visible to others).

## Endpoint

`POST /api/v1/query/{query_id}/unprivate`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| query_id | integer | Yes | The query ID |

## Example Request

```bash
curl -X POST "https://api.dune.com/api/v1/query/1234/unprivate" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```
