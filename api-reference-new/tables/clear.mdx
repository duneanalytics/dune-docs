# Clear Data

Remove all rows from a table while keeping the schema.

## Endpoint

`POST /api/v1/table/{table_name}/clear`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| table_name | string | Yes | The table name |

## Example Request

```bash
curl -X POST "https://api.dune.com/api/v1/table/my_table/clear" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```
