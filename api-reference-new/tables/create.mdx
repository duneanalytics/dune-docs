# Create Table

Create an empty table with a defined schema.

## Endpoint

`POST /api/v1/table/create`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| table_name | string | Yes | Name for the table |
| schema | array | Yes | Column definitions |

```json
{
  "table_name": "my_table",
  "schema": [
    {"name": "address", "type": "varchar"},
    {"name": "balance", "type": "double"}
  ]
}
```

## Example Request

```bash
curl -X POST "https://api.dune.com/api/v1/table/create" \
  -H "X-Dune-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"table_name": "my_table", "schema": [{"name": "address", "type": "varchar"}]}'
```
