# Pipelines

Get the status of query pipelines.

## Endpoint

`GET /api/v1/pipeline/{id}/status`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| id | string | Yes | The pipeline ID |

## Example Request

```bash
curl "https://api.dune.com/api/v1/pipeline/abc123/status" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```
