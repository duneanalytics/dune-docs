# Get Query Result

Retrieve the latest cached result for a query without re-executing.

## Endpoint

`GET /api/v1/query/{query_id}/results`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| query_id | integer | Yes | The query ID |

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| limit | integer | No | Max rows to return |
| offset | integer | No | Rows to skip |

## Example Request

```bash
curl "https://api.dune.com/api/v1/query/1234/results" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```

## Response

Same format as [Get Execution Result](/api-reference/executions/get-execution-result).

## Notes

- Returns the most recent successful execution result
- Faster than re-executing if cached results are acceptable
- May return stale data depending on last execution time
