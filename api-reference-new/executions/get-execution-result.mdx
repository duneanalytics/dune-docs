# Get Execution Result

Retrieve the results of a completed query execution as JSON.

## Endpoint

`GET /api/v1/execution/{execution_id}/results`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| execution_id | string | Yes | The execution ID |

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| limit | integer | No | Max rows to return (default: all) |
| offset | integer | No | Rows to skip |

## Example Request

```bash
curl "https://api.dune.com/api/v1/execution/01HKZJ2683PHF9Q9PHHQ8FW4Q1/results?limit=100" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```

## Response

```json
{
  "execution_id": "01HKZJ2683PHF9Q9PHHQ8FW4Q1",
  "query_id": 1234,
  "state": "QUERY_STATE_COMPLETED",
  "result": {
    "rows": [
      {"block_time": "2024-01-01T00:00:00Z", "value": 100},
      {"block_time": "2024-01-01T00:01:00Z", "value": 200}
    ],
    "metadata": {
      "column_names": ["block_time", "value"],
      "result_set_bytes": 1024,
      "total_row_count": 2
    }
  }
}
```
