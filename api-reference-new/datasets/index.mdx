# Datasets

Retrieve metadata about available datasets.

## Endpoint

`GET /api/v1/dataset/{name}`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| name | string | Yes | Dataset name (e.g., `ethereum.transactions`) |

## Example Request

```bash
curl "https://api.dune.com/api/v1/dataset/ethereum.transactions" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```

## Response

```json
{
  "name": "ethereum.transactions",
  "description": "All Ethereum transactions",
  "columns": [
    {"name": "block_time", "type": "timestamp"},
    {"name": "hash", "type": "varbinary"}
  ]
}
```
