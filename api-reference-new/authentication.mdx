# Authentication

Secure your API access with proper key management.

## API Keys

All API requests require an API key passed in the `X-Dune-API-Key` header.

```bash
curl -H "X-Dune-API-Key: YOUR_API_KEY" https://api.dune.com/api/v1/...
```

- Use environment variables

- Use separate keys for development and production
