```solidity
function simSalt() internal returns (bytes32 salt)
```

# Return Values

| Name | Type    | Description                                        |
| ---- | ------- | -------------------------------------------------- |
| salt | bytes32 | the salt if the ethereum transaction is a create2. |

# Notes

1. Can only be used in a CREATE2 transaction.
