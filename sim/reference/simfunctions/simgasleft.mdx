```solidity
function simGasLeft() internal returns (uint64 gas_)
```

# Return Values

| Name  | Type   | Description                                               |
| ----- | ------ | --------------------------------------------------------- |
| gas\_ | uint64 | the gas remaining of the Ethereum transaction/call/create |

# Notes

1. Can only be called in a transaction stop hook.
