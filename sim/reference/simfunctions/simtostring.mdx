# address

```solidity
function simToString(address data) internal returns (string string_)
```

## Parameters

| Name | Type    | Description          |
| ---- | ------- | -------------------- |
| data | address | the value to convert |

## Return Values

| Name     | Type   | Description                                     |
| -------- | ------ | ----------------------------------------------- |
| string\_ | string | the given value as a base 16 0x prefixed string |
