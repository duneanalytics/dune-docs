## Overview

Sim Real-Time APIs provide access to real-time blockchain simulation data, enabling:

- **Transaction Simulation** - Simulate transactions before execution
- **State Queries** - Query blockchain state in real-time
- **Event Streaming** - Stream blockchain events as they happen
