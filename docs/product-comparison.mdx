Dune delivers blockchain data via three core products: **Analytics Hub**, **Datashare**, and **Sim-Realtime APIs**. Each product serves different use cases, from ad-hoc analytics to real-time application development.

|---|---|---|---|

| **Query Speed** | ~3 seconds | N/A | N/A |
| **Engine** | Custom Trino (Small, Medium, Large) | N/A | N/A |

## Quick Selection Guide

<CardGroup cols={2}>
  <Card title="Analytics Hub" icon="chart-line" href="/web-app/overview">
    **Best for:** Your primary onchain data warehouse, analytic workflows, and data transformations

    Full SQL access with 100+ chains, flexible schemas, and compute-based pricing
  </Card>

  <Card title="Datashare" icon="database" href="/datashare/datashare">
    **Best for:** Data warehouse integration and interoperability

    Direct access via Snowflake, BigQuery, or Databricks with full historical data
  </Card>

  <Card title="Sim-Realtime APIs" icon="bolt" href="https://sim.dune.com">
    **Best for:** Building realtime wallets, portfolio trackers, and DeFi applications

    Sub-second response times with 60+ chains via REST APIs
  </Card>

  <Card title="Catalyst" icon="rocket" href="/catalyst/overview">
    **Best for:** Blockchain integration and data infrastructure

    Dune's toolkit for chains and protocols to get indexed and make their data accessible
  </Card>
</CardGroup>

<Note>
**Why is Catalyst not in the comparison tables?** Catalyst is Dune's blockchain integration toolkit designed for chains and protocols, not end-user data consumption. It powers how blockchains get indexed into Dune rather than how users query data, so it serves a fundamentally different purpose than Analytics Hub, Datashare, and Sim-Realtime APIs.
</Note>
