Dune delivers blockchain data via three core products: **Analytics Hub**, **Datashare**, and **Sim-Realtime APIs**.

| | <Icon icon="chart-line" size={16} /> [Analytics Hub](/web-app/overview) | <Icon icon="database" size={16} /> [Datashare](/datashare/datashare) | <Icon icon="bolt" size={16} /> [Sim-Realtime APIs](https://sim.dune.com) |
|:--|:--|:--|:--|

### Query & Architecture

| | | | |
|:--|:--|:--|:--|
| Query Infra | DuneSQL / Trino | Snowflake, BigQuery, Databricks | - |
| Query Speed | ~3 seconds | - | - |
| Engine | Custom Trino (Small, Medium, Large) | - | - |
| Architecture | Analytics | Analytics | Real-time |

### Data Coverage

| | | | |
|:--|:--|:--|:--|
| Freshness | Raw: ~20 min, Decoded: ~25 min, Curated: ~30 min - 6 hours | 1-24 hours | < 1 second |
| Blockchains | 115+ chains ([see more](/data-catalog/evm/overview)) | 115+ chains ([see more](/data-catalog/evm/overview)) | 60+ chains |
| Schemas | Raw, Decoded, Curated ([DEX Trades](/data-catalog/curated/dex-trades), [Prices](/data-catalog/curated/prices), [and more!](/data-catalog/curated/overview)) | Same as Analytics Hub | Curated (Balances, Activity, DeFi, Token Info) |

### Interface & Pricing

| | | | |
|:--|:--|:--|:--|
| Interface | [Dune App](https://dune.com), [Analytics API](/api-reference/api-overview), [External BI Tools](/access-methods/trino-connector/index) | [BigQuery](/datashare/datashare-bigquery), [Snowflake](/datashare/datashare-snowflake), [Databricks](/datashare/datashare-databricks) | APIs (Balances, Activity, Collectibles, DeFi Positions, Transactions, Token Info, Token Holders) |
| Pricing | Credits (compute-based) | Number of chains & schemas | API Call Volume (CUs) |

### Capabilities

| | | | |
|:--|:--|:--|:--|
| Best For | Your primary onchain data warehouse, Analytic workflows, [Transforming data](/access-methods/dbt-connector/index) | Data warehouse integration & Interoperability | Realtime apps, wallet tracking, portfolio apps |
| SQL Access | ✓ Full SQL | ✓ Full SQL | REST APIs only |
| Historical Data | ✓ Full historical | ✓ Full historical | Realtime |
| Authentication | Dune account, API key | Warehouse credentials | Sim account, API key |
