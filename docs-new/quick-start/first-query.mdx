## Step 2: Write Your First Query

Let's query recent Ethereum transactions:

```sql
SELECT 
    block_time,
    "from",
    "to",
    value / 1e18 as eth_value
FROM ethereum.transactions
WHERE block_time > now() - interval '1' hour
LIMIT 10
```
