## Snowflake

1. Go to [Dune on Snowflake Marketplace](https://app.snowflake.com/marketplace/providers/GZSVZMYSCO/Dune)
2. Click "Get" on any dataset for 30-day trial
3. Configure database name and roles
4. Start querying

<Warning>
You may need to click "Get" twice - once to copy data, then again to add to your Snowflake.
</Warning>

See [Snowflake guide](/datashare/datashare-snowflake) for data types and query examples.

## BigQuery

Email datashares-sales@dune.com or use our [Enterprise form](https://dune.com/enterprise)

**Include:**
- GCP Project ID
- Preferred region (US Central 1 or EU West 2)  
- Chains of interest

We'll set up trial access in your BigQuery project with technical support.

See [BigQuery guide](/datashare/datashare-bigquery) for data types and query examples.
