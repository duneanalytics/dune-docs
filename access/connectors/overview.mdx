# Connectors

Connect your existing tools and data pipelines to Dune's blockchain data.

## When to Use Connectors

Use connectors when you want to:
- Build data pipelines with dbt
- Query Dune from BI tools like Hex, Tableau, or Metabase
- Integrate with your existing data stack
