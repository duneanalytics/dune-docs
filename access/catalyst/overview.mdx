Complete data infrastructure solution for L1s, L2s, and blockchain networks.

## What is Catalyst?

Catalyst provides blockchain teams with:
- Full data indexing and processing
- Decoded contract tables
- Curated analytical datasets
- Public dashboards and analytics
- API access for ecosystem developers
