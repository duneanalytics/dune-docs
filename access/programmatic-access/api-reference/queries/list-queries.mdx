# List Queries

Retrieve a list of your saved queries.

## Endpoint

`GET /api/v1/query`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| limit | integer | No | Max results (default: 10) |
| offset | integer | No | Pagination offset |

## Example Request

```bash
curl "https://api.dune.com/api/v1/query?limit=10" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```

## Response

```json
{
  "queries": [
    {
      "query_id": 1234,
      "name": "My Query",
      "created_at": "2024-01-01T00:00:00Z"
    }
  ],
  "total_count": 50
}
```
