# Query Management

Create, read, update, and manage saved queries programmatically.

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | [List Queries](/api-reference/queries/list-queries) | List your queries |
| GET | [Get Query Pipeline](/api-reference/queries/get-query-pipeline) | Get pipeline info |
| POST | [Create](/api-reference/queries/create) | Create new query |
| PATCH | [Update](/api-reference/queries/update) | Update query |
| GET | [Read](/api-reference/queries/read) | Get query details |
| POST | [Archive](/api-reference/queries/archive) | Archive query |
| POST | [Unarchive](/api-reference/queries/unarchive) | Restore archived query |
| POST | [Private](/api-reference/queries/private) | Make query private |
| POST | [Unprivate](/api-reference/queries/unprivate) | Make query public |

## Query Object

```json
{
  "query_id": 1234,
  "name": "My Query",
  "description": "Query description",
  "query_sql": "SELECT * FROM ...",
  "is_private": false,
  "is_archived": false,
  "created_at": "2024-01-01T00:00:00Z",
  "updated_at": "2024-01-01T00:00:00Z"
}
```
