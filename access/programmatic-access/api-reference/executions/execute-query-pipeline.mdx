# Execute Query Pipeline

Execute a query as part of a pipeline workflow.

## Endpoint

`POST /api/v1/query/{query_id}/execute/pipeline`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| query_id | integer | Yes | The query ID to execute |
