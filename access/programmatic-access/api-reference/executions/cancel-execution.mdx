# Cancel Execution

Cancel a query execution that is currently running.

## Endpoint

`POST /api/v1/execution/{execution_id}/cancel`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| execution_id | string | Yes | The execution ID to cancel |

## Example Request

```bash
curl -X POST "https://api.dune.com/api/v1/execution/01HKZJ2683PHF9Q9PHHQ8FW4Q1/cancel" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```

## Response

```json
{
  "success": true
}
```

## Notes

- Only pending or executing queries can be cancelled
- Credits for cancelled queries are not charged
