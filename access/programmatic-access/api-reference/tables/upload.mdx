# Upload CSV

Upload a CSV file to create or replace a table.

## Endpoint

`POST /api/v1/table/upload/csv`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Request

Send a multipart form with:
- `table_name`: Name for the table
- `file`: The CSV file

## Example Request

```bash
curl -X POST "https://api.dune.com/api/v1/table/upload/csv" \
  -H "X-Dune-API-Key: YOUR_API_KEY" \
  -F "table_name=my_addresses" \
  -F "file=@addresses.csv"
```

## Response

```json
{
  "success": true,
  "table_name": "dune.username.my_addresses"
}
```
