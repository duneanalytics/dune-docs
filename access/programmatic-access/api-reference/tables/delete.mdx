# Delete Table

Permanently delete a table and all its data.

## Endpoint

`DELETE /api/v1/table/{table_name}`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| table_name | string | Yes | The table name |

## Example Request

```bash
curl -X DELETE "https://api.dune.com/api/v1/table/my_table" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```

## Warning

This action is irreversible. All data will be permanently deleted.
