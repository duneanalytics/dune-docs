## Endpoints

### Executions and Results
Execute queries and retrieve results.

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | [Execute SQL](/api-reference/executions/execute-sql) | Run raw SQL |
| POST | [Execute Query](/api-reference/executions/execute-query) | Run saved query |
| GET | [Get Status](/api-reference/executions/get-execution-status) | Check execution status |
| GET | [Get Result](/api-reference/executions/get-execution-result) | Retrieve results |
| POST | [Cancel](/api-reference/executions/cancel-execution) | Cancel execution |

### Query Management
Create and manage saved queries.

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | [List Queries](/api-reference/queries/list-queries) | List your queries |
| POST | [Create](/api-reference/queries/create) | Create new query |
| GET | [Read](/api-reference/queries/read) | Get query details |
| PATCH | [Update](/api-reference/queries/update) | Update query |

### Data Uploads
Upload your own data to Dune.

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | [Upload CSV](/api-reference/tables/upload) | Upload CSV file |
| POST | [Create Table](/api-reference/tables/create) | Create empty table |
| POST | [Insert Data](/api-reference/tables/insert) | Insert rows |
| DELETE | [Delete Table](/api-reference/tables/delete) | Delete table |
