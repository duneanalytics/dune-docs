# Refresh Materialized View

Trigger a manual refresh of a materialized view.

## Endpoint

`POST /api/v1/materialized-view/{name}/refresh`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| name | string | Yes | The view name |

## Example Request

```bash
curl -X POST "https://api.dune.com/api/v1/materialized-view/my_view/refresh" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```
