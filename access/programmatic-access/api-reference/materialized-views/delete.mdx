# Delete Materialized View

Permanently delete a materialized view.

## Endpoint

`DELETE /api/v1/materialized-view/{name}`

## Authentication

Requires API key in header: `X-Dune-API-Key`

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| name | string | Yes | The view name |

## Example Request

```bash
curl -X DELETE "https://api.dune.com/api/v1/materialized-view/my_view" \
  -H "X-Dune-API-Key: YOUR_API_KEY"
```
