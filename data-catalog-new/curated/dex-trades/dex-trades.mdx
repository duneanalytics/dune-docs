# dex.trades

Aggregated DEX swap data across all supported chains and protocols.

## Quick Example

```sql
SELECT * FROM dex.trades 
WHERE block_time > now() - interval '7' day
LIMIT 100
```

## Example Queries

### Daily Volume by DEX

```sql
SELECT 
    date_trunc('day', block_time) as day,
    project,
    SUM(amount_usd) as volume_usd
FROM dex.trades
WHERE block_time > now() - interval '30' day
GROUP BY 1, 2
ORDER BY 1 DESC, 3 DESC
```

### Top Trading Pairs

```sql
SELECT 
    token_pair,
    COUNT(*) as trade_count,
    SUM(amount_usd) as total_volume
FROM dex.trades
WHERE block_time > now() - interval '7' day
GROUP BY 1
ORDER BY 3 DESC
LIMIT 20
```
