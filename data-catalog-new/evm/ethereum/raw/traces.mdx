# ethereum.traces

Internal transactions (traces) on Ethereum mainnet.

## Quick Example

```sql
SELECT * FROM ethereum.traces
WHERE block_time > now() - interval '1' hour
  AND success = true
LIMIT 100
```

## Schema

| Column | Type | Description |
|--------|------|-------------|
| block_time | timestamp | Block timestamp |
| block_number | bigint | Block number |
| tx_hash | varbinary | Transaction hash |
| from | varbinary | Caller address |
| to | varbinary | Called address |
| value | uint256 | ETH value in wei |
| type | varchar | Call type (call, delegatecall, etc.) |
| success | boolean | Whether the call succeeded |
