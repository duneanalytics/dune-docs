## What is Somnia?

Somnia is a high-performance, cost-efficient EVM-compatible Layer 1 blockchain capable of processing over 1,000,000 transactions per second (TPS) with sub-second finality. It is suitable for serving millions of users and building real-time mass-consumer applications like games, social applications, metaverses, and more, all fully on-chain.

## Key Features

- **Ultra-High Performance**: Capable of processing over 1,000,000 TPS with sub-second finality
- **EVM-Compatible**: Full compatibility with Ethereum smart contracts and tooling
- **Mass-Consumer Focus**: Designed for real-time applications serving millions of users
- **Cost-Efficient**: Optimized for high-throughput, low-cost transactions
- **Gaming & Metaverse Ready**: Specialized infrastructure for gaming and metaverse applications
- **Social Applications**: Built to support large-scale social platforms and applications

## Performance Benchmarks

- **1,050,000 TPS**: Achieved with ERC-20 transfers between several hundred thousand accounts
- **50,000 Uniswaps per second**: Running Uniswap V3 with several hundred thousand accounts trading one LP pool
- **300,000 NFT mints per second**: Supporting 100,000 accounts minting NFTs simultaneously

## Ecosystem Highlights

- High-performance infrastructure for mass-consumer applications
- Gaming and metaverse platform capabilities
- Social application support at scale
- Real-time transaction processing
- Support for various application types including:
  - Games
  - Social applications
  - Metaverses
  - Real-time consumer applications
  - High-throughput DeFi protocols
