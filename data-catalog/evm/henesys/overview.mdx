## What is Henesys?

Henesys is Avalanche's Layer 1 blockchain that powers the NEXPACE ecosystem and the MapleStory Universe. Built by Nexon's blockchain subsidiary, Henesys provides a secure, scalable platform for Web3 gaming, enabling player-owned economies, decentralized governance, and cross-chain interoperability through Chainlink's CCIP integration.

## Key Features

- **Gaming-Focused Infrastructure**: Specialized blockchain for Web3 gaming and NFT ecosystems
- **Avalanche L1**: Built on Avalanche for high performance and scalability
- **Player-Owned Economies**: Enables true ownership of in-game assets through NFTs
- **Fusion-Fission Model**: Dynamic NFT/item supply balancing mechanism
- **Cross-Chain Interoperability**: Chainlink CCIP integration for secure NFT portability across games
- **Permissionless Chain**: Open blockchain layer for deploying dApps and smart contracts

## Ecosystem Highlights

- MapleStory Universe integration with blockchain technology
- NFT minting and trading for in-game assets
- Synergy Apps ecosystem with open APIs for third-party developers
- Decentralized governance through NXPC token
- Support for various gaming use cases including:
  - In-game item tokenization
  - Player-to-player trading
  - Cross-game asset portability
  - Prediction markets and dashboards
  - Community-driven tooling
