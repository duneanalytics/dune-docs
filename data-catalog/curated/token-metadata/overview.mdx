Dune provides comprehensive token metadata that enables identification and analysis of digital assets across blockchain networks. Token metadata includes essential information such as token names, symbols, decimals, and contract addresses, which are fundamental for any token-related analysis.

## Key Features

- **Multi-Chain Coverage**: Token metadata across EVM and Solana ecosystems
- **Automatic Detection**: New tokens are automatically added when detected on-chain
- **Standardized Schema**: Consistent data structure for cross-chain analysis
- **Essential Token Information**: Names, symbols, decimals, and contract addresses
