## Table Description
Block-level data for block rewards on THORChain. This table is partitioned by day.

## Column Descriptions
| **Column** | **Type** | **Description** |
|------------|----------|-----------------|
| **day** | date | Date of the record |
| **liquidity_fee** | varchar | Identifier |
| **block_rewards** | varchar | Block Rewards |
| **earnings** | varchar | Earnings |
| **bonding_earnings** | varchar | Bonding Earnings |
| **liquidity_earnings** | varchar | Identifier |
| **avg_node_count** | varchar | Avg Node Count |
| **_inserted_timestamp** | timestamp | Timestamp of the record |
| **inserted_timestamp** | timestamp | Time when the record was inserted |
| **modified_timestamp** | timestamp | Time when the record was last modified |
