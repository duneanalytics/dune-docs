## Table Description
Daily statistics for Continuous Liquidity Pools (CLPs) on THORChain. CLPs are THORChain's advanced AMM mechanism that allows for native asset swaps without wrapped tokens. This table tracks daily pool statistics including liquidity, swaps, fees, and rewards. This table is partitioned by day.
