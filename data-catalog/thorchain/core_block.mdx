## Table Description
Contains block-level data from the THORChain blockchain including block headers, timestamps, and block statistics. This table is partitioned by block_month.

This table is created using [this SQL](https://github.com/duneanalytics/spellbook/blob/main/dbt_subprojects/daily_spellbook/models/thorchain/gold/thorchain_core_block.sql).

## Column Descriptions

| **Column**                    | **Type**      | **Description**                                   |
|-------------------------------|---------------|---------------------------------------------------|
| **dim_block_id**              | varchar       | Surrogate key for the block                       |
| **block_id**                  | bigint        | Block height/ID                                  |
| **block_time**                | timestamp     | Timestamp of when the block was created           |
| **block_timestamp**           | timestamp     | Timestamp of when the block was created           |
| **block_date**                | date          | Date of when the block was created                |
| **block_hour**                | timestamp     | Hour of when the block was created                |
| **block_week**                | timestamp     | Week of when the block was created                |
| **block_month**               | timestamp     | Month of when the block was created               |
| **block_quarter**             | int           | Quarter of when the block was created             |
| **block_year**                | int           | Year of when the block was created                |
| **block_dayofmonth**          | int           | Day of month when the block was created           |
| **block_dayofweek**           | int           | Day of week when the block was created            |
| **block_dayofyear**           | int           | Day of year when the block was created            |
| **timestamp**                 | bigint        | Raw timestamp from the block                       |
| **hash**                      | varchar       | Hash of the block                                 |
| **agg_state**                 | varchar       | Aggregated state information                       |
| **inserted_timestamp**        | timestamp     | Time when the record was inserted                  |
| **modified_timestamp**        | timestamp     | Time when the record was last modified            |
