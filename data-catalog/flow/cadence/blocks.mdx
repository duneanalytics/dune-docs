## Table Description

Contains block-level data from the Flow Cadence blockchain including block headers, timestamps, and network fee information.

## Column Descriptions

| Column | Type | Description |
|--------|------|-------------|
| block_date | date | The date of the block |
| timestamp | timestamp | Timestamp when the block was created |
| block_hash | string | Unique hash identifier for the block |
| block_height | bigint | The block number/height |
| parent_block_hash | string | Hash of the parent block |
| slot | bigint | Slot number |
| difficulty | decimal(38,9) | Block difficulty |
| total_difficulty | decimal(38,9) | Cumulative difficulty |
| network_fee_used | decimal(38,9) | Network fee consumed in this block |
| network_fee_limit | decimal(38,9) | Maximum network fee limit for the block |
| chain_specific | string | Chain-specific metadata in JSON format |
| _updated_at | timestamp | Timestamp when the record was last updated |
| _ingested_at | timestamp | Timestamp when the record was ingested |
