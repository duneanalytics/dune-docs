## Table description

This is an unnested table of [`solana.transactions`](/data-catalog/solana/transactions). There can be multiple instructions in a transaction, so having an exploded view here will make it a little easier to work with the data. **This table mainly exists as the base for [Solana decoded tables](/data-catalog/solana/idl-tables), which you should be using instead.**
